## [1.2.0]

- Added `public` context kind
- Extended property facets: `functional`, `selfonly`, `authoronly`, `maxExclusive`, `minExclusive`, `whiteSpace`, `totalDigits`, `fractionDigits`, `messageProperty`, `mediaProperty`, `readableName`, `regexp`, `setting`
- Extended perspective parts: `without`, `default`, `no roleverbs`
- Added state keywords: `in state`, `subject state`, `object state`, `context state`
- Added assignment keywords: `bind`, `unbind`, `unbind_`, `bind_`, `move`, `create_ context ... bound`, `callEffect`, `callDestructiveEffect`, `callExternal`
- Extended operators: `fills`, `filledBy`, `letE`, `letA`, `union`, `intersection`, `orElse`, `binding`, `this`, `me`, `indexedName`, `publicrole`, `publiccontext`, `isInState`, `contextinstance`, `roleinstance`, `specialisesRoleType`, `roleTypes`, `contextType`, `roleType`, `modelname`, `returns`, `translate`
- Extended property range types: `Date`, `Time`, `File`, `Email`, `MarkDown`
- Added screen element keywords: `screen`, `tab`, `row`, `column`, `form`, `markdown`, `table`, `chat`, `messages`, `media`, `who`, `what`, `where`, `master`, `detail`, `when`, `fillfrom`
- Extended time constants with singular/plural variants
